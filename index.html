Key Finding · MS Capstone Thesis · Rutgers University 2025

A persistence baseline outperformed the VAR model.
Here is why that is the correct result.

For my MS capstone I built a compositional forecasting pipeline for U.S. state-level electricity generation mix of 50 states, 3 fuel categories, 1990–2023. I applied centered log-ratio (CLR) transformation to handle the compositional constraint, then fit per-state VAR models with AIC-based lag selection.

Then I ran the simplest possible baseline: next year's mix equals this year's mix. The baseline won across every fuel category.

Why this is the right finding, not a failure: Annual electricity generation shares change slowly. With nearly 30 observations per state, a multivariate VAR overfits. The practical implication is clear: to beat persistence you need exogenous covariates, be it fuel price shocks, Renewable Portfolio Standards, carbon regulations, not a more complex autoregressive model. That is what I would build next.

I'm an MS Data Science graduate from Rutgers (Statistics Track, December 2025) based in Jersey City, NJ. My work sits at the intersection of quantitative modeling and domain expertise: specifically energy markets and credit risk, two fields where model evaluation matters as much as model building.

Getting that forecast right requires both statistical rigor and genuine domain context.

Code sample: PDO scorecard mapping
# Calibrated P(good) → interpretable scorecard score
# PDO=20 pts to double odds | SCORE0=600 at ODDS0=50
import numpy as np

def prob_to_score(p_good, score0=600, odds0=50, pdo=20):
    odds   = np.array(p_good) / (1 - np.array(p_good) + 1e-9)
    factor = pdo / np.log(2)
    offset = score0 - factor * np.log(odds0)
    return np.round(offset + factor * np.log(np.maximum(odds, 0.001))).astype(int)

# P(good)=0.09 → score 421 → DECLINE
# P(good)=0.86 → score 650 → APPROVE
